# Sales-Data-Analysis-Using-Python
Sales Data Analysisi

Sales Performance Analysis of an E-commerce Store
This project analyzes e-commerce sales data to uncover revenue trends, top products, and customer behavior using Python, Pandas, and NumPy. The analysis includes data cleaning, exploratory data analysis (EDA), and visualization to deliver actionable business insights.

Project Overview
Analyze 12 months of e-commerce sales data.

Identify top-selling products and monthly sales trends.

Visualize findings using line plots, bar charts, and stacked bar charts.

Tools Used
Python

Pandas (data wrangling)

NumPy (basic calculations)

Matplotlib and Seaborn (visualization)

Key Features
Import and clean raw sales data.

Calculate total sales and identify top products.

Visualize monthly sales trends and product performance.

Highlight seasonal peaks and high-performing categories.

How to Run
Clone the repository.

Place sales_data.csv in the project directory.

Run sales_analysis.py in your Python environment.
